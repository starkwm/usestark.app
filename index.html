Stark — Window Manager Tools for macOS

The goal of the Stark organisation is to create productivity tools for macOS.

Stark

Power your window management with JavaScript

Stark is a macOS window manager built using Swift. It utilises the JavaScriptCore framework. Stark exposes JavaScript objects that form the API for building your configuration. Stark sits in the menu bar out of the way, running in the background.

You can get started with Stark by learning how to install, and configure. Documentation coming soon..

skbd

Stark Key Bind Daemon

skbd is a daemon service that runs in the background built using Swift. skbd allows you to bind abitrary keyboard shortcuts to shell commands.

You can get started with skbd by learning how to install, and configure. Documentation coming soon..

Documentation | Source

swm

Stark Window Manager

swm is the spiritual successor to Stark. It is a window manager built using Swift, that runs completely in the background. It uses IPC to talk with the running process to trigger window management functions. When used with skbd, shortcut keys can trigger different window management functions.

Documentation will become available with the first release.
