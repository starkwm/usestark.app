Documentation for Stark

Power your window management with JavaScript

Stark is a macOS window manager built using Swift. It utilises the JavaScriptCore framework. Stark exposes JavaScript objects that form the API for building your configuration. Stark sits in the menu bar out of the way, running in the background.

Installation

The quickest way to get stark installed is using Homebrew.

$ brew install starkwm/formulae/stark

Configuration

The stark configuration file is in ~/.config/stark/stark.js.
