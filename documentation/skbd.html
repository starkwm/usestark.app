Documentation for skbd

Stark Key Bind Daemon

skbd is a daemon service that runs in the background built using Swift. skbd allows you to bind abitrary keyboard shortcuts to shell commands.

Installation

The quickest way to get skbd installed is using Homebrew.

$ brew install starkwm/formulae/skbd

When installing with Homebrew, you can use brew services start skbd. This launches and keeps the skbd process running.

You can build from source. This will require the latest Xcode.

$ git clone git@github.com:starkwm/skbd.git
$ cd skbd
$ make

You will have to create your own Launch Agent .plist to keep running skbd in the background.

Configuration

The skbd configuration file is in ~/.config/skbd/skbdrc by default. It can be overridden with the -c/--config flag.

Key binds are 1 or more modifiers and 1 key, followed by a shell command.

cmd + shift - b: open -a Alacritty

The above example binds the modifiers cmd and shift with the b key. It will launch the Alacritty terminal.

Use skbd -r to have skbd reload the configuration after making modifications.

Modifiers

Below is a list of supported modifier keys (and alternative spellings).

  • shift
  • ctrl/control
  • alt/opt/option
  • cmd/command
  • hyper (alias for cmd+opt+shift+ctrl)

Commands

The shell specified in the $SHELL environment variable executes the commands. It will fallback to /bin/bash if not set.

Currently the command will not use your shell configuration. You may need to use full paths for binaries that do not appear in the default path environment variable. We are looking into a way to solve this in a future release.
